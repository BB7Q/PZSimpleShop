[center][h1]SimpleShop - 简单商店[/h1][/center]

[h2]简介[/h2]
SimpleShop 是一个轻松实用的购物mod，为玩家提供便捷的商店界面来购买各种游戏物品。本mod参考了CoxisShop并进行了简化设计，专门专注于物品购买功能，让您在末世中能够更轻松地获取所需物资。商品列表完全兼容其他mod物品，即使物品不存在也没关系，mod会自动检测确保只显示当前游戏中存在的物品。设计简洁直观，方便与其他mod配合使用，为您带来更好的游戏体验。

[h2]主要功能[/h2]
[list]
    [*] 商店界面 - 滚动列表显示物品，支持分类筛选和搜索功能
    [*] 货币系统 - 通过击杀僵尸获得金钱
    [*] 死亡继承 - 玩家死亡后保留全部金钱
    [*] 右键菜单 - 在世界界面或物品栏中右键点击，选择"打开商店"
    [*] 多语言支持 - 内置中英文界面
    [*] 可配置 - 支持自定义初始金钱和物品价格
[/list]

[h2]如何使用[/h2]
[list]
    [*] 打开商店：在世界界面或物品栏中右键点击，选择"打开商店"
    [*] 购买物品：打开商店界面，点击要购买的物品，然后点击"购买"按钮
    [*] 搜索物品：使用搜索框快速查找所需物品
    [*] 分类筛选：使用下拉菜单按类别浏览物品
    [*] 获得金钱：击杀僵尸可获得金钱奖励，新角色开始时有初始资金
[/list]

[h2]配置选项[/h2]
[list]
    [*] 初始金钱：100（可在SimpleShopBasic.lua中修改）
    [*] 击杀奖励：每杀死一个僵尸获得10金钱
    [*] 可购买物品：包括基础生存物资、医疗用品、工具、武器、建筑材料和弹药
[/list]

[b]自定义物品[/b]
商店支持添加来自其他mod的物品，即使物品不存在也没关系，mod会自动检测只显示存在的物品。要添加新物品到商店，只需在SimpleShopItems.lua中添加新行：
[code]
["物品ID"] = 价格, -- 物品名称
[/code]

[b]创建自定义配置模组[/b]
为了避免在模组更新时丢失配置，建议创建一个单独的配置模组：

1. 创建一个新的模组文件夹
2. 在其中创建以下文件结构：
[code]
MyShopConfig/
├── 42/
│   └── mod.info
└── common/
    └── media/
        └── lua/
            └── shared/
                ├── SimpleShopBasic.lua
                └── SimpleShopItems.lua
[/code]

注意：为了兼容游戏的B42版本，需要创建42版本的文件夹。mod.info文件只需要放在42文件夹内。

3. 在mod.info中添加：
[code]
name=My Shop Config
id=MyShopConfig
description=Custom configuration for SimpleShop
poster=poster.png
depends=SimpleShop
[/code]

4. 在你的配置文件中覆盖所需值：
[code]
-- SimpleShopBasic.lua
SimpleShopBasic = {
    ["initialMoney"] = 100, -- 初始金钱
    ["zombieKillAmount"] = 10, -- 每击杀一个僵尸获得的金钱
}

-- SimpleShopItems.lua
SimpleShopItems = {
    -- 你的自定义物品价格
    ["Base.Bread"] = 50, -- 面包
    ["Base.CustomItem"] = 250, -- 自定义物品
}
[/code]

5. 确保在游戏启动器中，你的配置模组位于SimpleShop之后加载

这种方法可以让你在保持模组更新的同时保留自定义配置。

[h2]API接口[/h2]
SimpleShop提供了以下API，供其他mod调用以修改玩家金钱：

[h3]获取玩家金额[/h3]
[code]
local money = SimpleShop.GetPlayerMoney(playerIndex)
[/code]
获取指定玩家的当前金额，不指定playerIndex则获取当前玩家金额。

[h3]增加玩家金额[/h3]
[code]
SimpleShop.AddPlayerMoney(amount, playerIndex)
[/code]
为指定玩家增加金额，amount必须为正数。

[h3]减少玩家金额[/h3]
[code]
SimpleShop.RemovePlayerMoney(amount, playerIndex)
[/code]
从指定玩家扣除金额，amount必须为正数且玩家余额必须足够。

[h3]检查玩家金额[/h3]
[code]
SimpleShop.HasEnoughMoney(amount, playerIndex)
[/code]
检查指定玩家是否有足够金额。

这些方法可用于实现任务奖励、自定义商店等需要修改玩家金钱的功能。

[h2]Introduction[/h2]
SimpleShop is a practical and user-friendly shopping mod that provides players with a convenient shop interface to purchase various items in the game. Based on CoxisShop and redesigned with simplicity in mind, this mod focuses solely on purchasing functionality, helping you easily obtain essential supplies in the post-apocalyptic world. The item list fully supports items from other mods, and even if items don't exist, it doesn't matter - the mod will automatically detect and ensure only existing items are displayed. Its straightforward design makes it compatible with other mods, enhancing your overall gaming experience.

[h2]Key Features[/h2]
[list]
    [*] Shop interface - Scrollable list display with category filtering and search functionality
    [*] Currency system - Earn money by killing zombies
    [*] Death inheritance - Keep all money after player death
    [*] Right-click menu - Right-click in the world or inventory and select "Open Shop"
    [*] Multi-language support - Built-in Chinese and English interface
    [*] Configurable - Support for custom initial money and item prices
[/list]

[h2]How to Use[/h2]
[list]
    [*] Open shop: Right-click in the world or inventory and select "Open Shop"
    [*] Purchase items: Open the shop interface, click the item you want to buy, then click the "Buy" button
    [*] Search items: Use the search box to quickly find items you need
    [*] Category filter: Use the dropdown menu to browse items by category
    [*] Earn money: Get money rewards by killing zombies, new characters start with initial funds
[/list]

[h2]Configuration Options[/h2]
[list]
    [*] Initial money: 100 (can be modified in SimpleShopBasic.lua)
    [*] Kill reward: Get 10 money for each zombie killed
    [*] Purchasable items: Including basic survival supplies, medical supplies, tools, weapons, building materials, and ammunition
[/list]

[h2]API Interface[/h2]
SimpleShop provides the following APIs for other mods to call to modify player money:

[h3]Get Player Money[/h3]
[code]
local money = SimpleShop.GetPlayerMoney(playerIndex)
[/code]
Get the current amount of money for the specified player, or get the current player's money if playerIndex is not specified.

[h3]Add Player Money[/h3]
[code]
SimpleShop.AddPlayerMoney(amount, playerIndex)
[/code]
Add money to the specified player, amount must be a positive number.

[h3]Remove Player Money[/h3]
[code]
SimpleShop.RemovePlayerMoney(amount, playerIndex)
[/code]
Deduct money from the specified player, amount must be a positive number and the player must have sufficient balance.

[h3]Check Player Money[/h3]
[code]
SimpleShop.HasEnoughMoney(amount, playerIndex)
[/code]
Check if the specified player has enough money.

These methods can be used to implement features such as quest rewards, custom shops, or any functionality that requires modifying player money.
